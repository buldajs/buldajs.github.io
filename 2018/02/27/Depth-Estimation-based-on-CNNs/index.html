Depth Map Prediction from a Single Image using a Multi-Scale Deep Network

这篇中了NIPS2014，是第一篇CNN-based来实现单目深度估计的文章。

Contribution

实现了第一个CNN-based Multi-scale Network.

不同于如今网络中Multi-scale features combined做法，这里的Multi-scale是指整个网络分为两个scale的网络来做DepthMap的估计。分别是Global Coarse-Scale Network和Local Fine-Scale Network。

  • Global Coarse-Scale Network

    网路结构类似于AlexNet，得到一个low-resolution的coarse的depth map,将输出接于后续网络继续优化.

  • Local Fine-Scale Network

    将前者的输出与rgb经过conv后concat，再通过一些conv层去refine前者的输出得到最后的refined depth map.

两个网络的训练是分开的，先train前者，后固定前者train后者。

Predicting depth, surface normals and semantic labels with a common multi-scale convolutional architecture

这篇中了ICCV2015，也是上一篇作者David Eigen的工作，是一个multi-task的network.

Contribution

这篇在上一篇的网络结构基础上做了一些部分改进。

网络从两个scale增加到了三个scale，Scale1用VGG替换AlexNet，这篇没细看，有错请指出～

Learning Fine-Scaled Depth Maps from Single RGB Images

16年7月挂在arxiv上的工作,在上一篇的基础上做了一些改进。

Contribution

  • 在ICCV2015的基础上引入multi-scale间的skip connections,加速网络的收敛。

  • 考虑在做Data Augmentation时相同数据生成的train data之间的相关性。因此构建了基于a set of transformed images的loss的约束。

Joint Semantic Segmentation and Depth Estimation with Deep Convolutional Networks

arxiv1604的工作，Joint分割和单目深度估计的文章，看完没什么感觉，有兴趣的可以看看～

Deeper Depth Prediction with Fully Convolutional Residual Networks

IEEE 3D Vision 2016的文章，使用ResNet的结构来实现Depth Estimate.

Contribution

  • 使用ResNet来实现Depth Estimation
  • 针对unpooling+deconv的up-convolution单元，提出了基于残差思想的up-projection单元，并提出了两者的Fast版本来加速计算。

Decoder部分的四个版本

  • up-convolution

    基本的unpooling+convolution

  • up-projection

    引入了残差的up-convolution单元

  • fast-up-convolution

    unpooling+5*5的conv可以用4个更小的conv来实现，从而加速.

  • fast-up-projection

    同上，替换unpooling+5*5的conv.

Unsupervised Monocular Depth Estimation With Left-Right Consistency

CVPR2017的工作.

这篇没细看,思想是loss不是直接用的depth的loss,而是用估计出来的depthmap来做左右图的匹配.

利用匹配后rgb图像强度偏差构建loss,所以说是unsupervised.

Semi-Supervised Deep Learning for Monocular Depth Map Prediction

CVPR2017的工作.

这篇是semi-supervised,网络结构基本用的是上面的FCRN,主要在loss上有改进。

Loss function

Loss一共分为Supervised loss,Unsupervised loss,Regularization loss.

  • Unsupervised loss

    和上一篇unsupervised相同思想，用的也是网络inference得到depthmap，做左右rgb图像匹配构建loss.

  • Supervised loss

    因为用的是kitti数据，有雷达图像数据配准后的sparse的深度图，将这些sparse的depth values作为seed点，也引入了loss中。

  • Regularization loss

    添加了gradient的Regularization loss作为约束。

所以个人感觉这篇文章是上两篇文章的结合。

Multi-Scale Continuous CRFs as Sequential Deep Networks for Monocular Depth Estimation

CVPR2017工作，xiaogang wang他们的成果.

这篇主要是CNN和Graphical Model的结合。

Motivation

  • Multi-scale

    在CNN解决pixel级的classification/regression问题时，引入multi-scale的信息可以更好结合low-level和high-level的feature。

  • Continuous CRFs

    在semantic segmentation问题中，CRFasRNN中将CRF的mean-field解法展开用RNN网络的结构实现，使得CNN+CRF可以进行end-to-end的优化.这是针对discrete domain的问题，而对于continuous domain的regression问题，还没有这样翻译成网络的CRF层的存在。

  • Graphical Model

    图模型有很好的表达能力，可以与CNN模型的特征表达结合，得到更好的效果。

因此作者提出了Multi-scale CRFs和Cascade CRFs两个模块。

具体推导不展开，有兴趣的可以看论文，两个CRFs的最小单元可以用一个C-MF block来实现。

总结

对于单目的深度估计问题，在网络结构上个人认为和semantic segmentation那套区别不大。

semantic segmentation是pixel-wise classfication问题，而单目depth estimation是pixel-wise regression问题。

区别在于用的loss的差异.

对于depth estimation还可以利用rgb图像和depth图像之间结构的关联性以及其他特性去构建loss,来提升性能。

个人浅显的理解，有问题请指出～
